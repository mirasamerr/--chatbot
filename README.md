# Chatbot - لحظة
An AI-powered chatbot for لحظة, an online store for old style handwritten letters. Built with Python and Groq API.

## What it does
- Answers questions about لحظة brand identity
- Remembers the full conversation history
- Tracks how many messages were sent per session
- Refuses to answer pricing questions with a fun response
- Greets users personally by name

## Tech Stack
- Python
- Groq API (LLaMA 3.1)

## How to run
1. Clone this repository
2. Install dependencies:
```
   pip install groq
```
3. Add your Groq API key in the code
4. Run the chatbot:
```
   python lahza_chatbot.py
```
